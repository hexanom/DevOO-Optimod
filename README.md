Usage
-----

### From command-line

#### OSX/Linux

```shell
# start the app
./gradlew run
# run the tests
./gradlew check
# package the app
./gradlew distZip
```

#### Windows

```shell
# start the app
gradlew.bat run
# run the tests
gradlew.bat check
# package the app
gradlew.bat distZip
```
