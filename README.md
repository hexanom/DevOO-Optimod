Usage
-----

### Run

#### OSX/Linux

```shell
./gradlew run
```

#### Windows

```shell
gradlew.bat run
```

### Test

#### OSX/Linux

```shell
./gradlew test
```

#### Windows

```shell
gradlew.bat test
```
